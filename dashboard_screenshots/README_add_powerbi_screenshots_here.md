## How to Take Screenshots in Power BI

1. Build the dashboard following `docs/powerbi_dashboard_guide.md`
2. For full dashboard: use **File → Export → Export to PDF** or take a full-screen screenshot
3. For individual panels: right-click each visual → **Export data** (table view) or use Windows Snipping Tool
4. Save at minimum 1920×1080 resolution, PNG format
5. Name files exactly as shown in the table above
